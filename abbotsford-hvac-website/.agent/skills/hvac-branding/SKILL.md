# HVAC Branding Skill

This skill provides a foundation for creating premium digital experiences for HVAC businesses.

## Design Philosophy

- **Trust and Reliability**: Use sturdy, professional typography.
- **Precision**: Clean lines and technical details.
- **Comfort**: Visual cues related to temperature control and air quality.

## Color Palette

- **Primary Blue (Cooling)**: `#1A73E8`
- **Primary Red (Heating)**: `#D93025`
- **Neutral Silver (Equipment)**: `#E8EAED`
- **Dark Slate (Professionalism)**: `#202124`
